# Viikko 3.

Tällä viikolla, ehdin kokeilla käytännöllisempää tapaa soittaa eri sointuja: ensimmäinen sointu on vapaasti soiva jolloin kaksi muuta voi ottaa "helposti" painamalla kahdesta eri kohtaa kaikki kielet alas.
Ongelmana on että kielten alaspainamiseen vaadittava voima sai koko moottorin nousemaan sijoiltaan ja tämän korjaaminen taas vaatii uutta lähestymistapaa. Työ siis tältä osin kesken, mutta edistystäkin on havaittavissa.
